Utility Helpers
===============

Overview
--------
The :mod:`deladect.utils` module provides small geometry helpers used in crack
post-processing and reporting workflows.

Crack helpers in this module use ``[row, col]`` coordinate ordering.

Available helpers
-----------------
- :func:`deladect.utils.crack_mid_point` - midpoint of one crack segment.
- :func:`deladect.utils.crack_length` - Euclidean segment length.
- :func:`deladect.utils.crack_px_mm` - coordinate conversion from pixels to mm.

Example
-------

.. code-block:: python

    from deladect.utils import crack_length, crack_mid_point, crack_px_mm

    # [row, col] endpoints
    crack = [[10.0, 20.0], [35.0, 24.0]]

    length_px = crack_length(crack)
    midpoint = crack_mid_point(crack)
    crack_mm = crack_px_mm([crack], scale=41.03328366)

    print(length_px, midpoint, crack_mm)

Common usage notes
------------------
- ``crack_mid_point`` returns ``[row_mid, col_mid]``.
- ``crack_length`` is Euclidean distance in pixel units.
- ``crack_px_mm`` scales coordinates only (it does not change coordinate order).
